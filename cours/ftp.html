Le déploiement FTP via github actions

Créer un dossier .github à la racine du projet, ensuite dedans un dossier workflows, puis un fichier deploy.yml

Mettez le texte suivant, et modifiez par vos informations


        name: Deploy to InfinityFree

        on:
        push:
            branches:
            - main  # Change ici si ton site est sur une autre branche

        jobs:
        ftp-deploy:
            name: Upload via FTP
            runs-on: ubuntu-latest
            steps:
            - name: Checkout repo
                uses: actions/checkout@v3

            - name: Deploy files via FTP
                uses: SamKirkland/FTP-Deploy-Action@v4.3.4
                with:
                server: ftpupload.net
                username: ${{ secrets.FTP_USERNAME }} 
                password: ${{ secrets.FTP_PASSWORD }}
                local-dir: ./
                server-dir: /tominou.tech/htdocs/

Ajoutes des secrets à votre repo github: FTP_USERNAME, FTP_PASSWORD
